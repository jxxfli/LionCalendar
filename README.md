#### 因项目需求写的库，有类似功能的可以clone下来改改😁
在主项目app的build.gradle中依赖
```gradle
dependencies {
	...
	implementation 'com.lion:LionCalendar:1.0.2'
}
```

没有架构可言
### 支持3种选择模式：月份选择模式、周日选择模式和普通模式
